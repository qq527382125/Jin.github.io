自动格式化代码

第一种：用默认的第一次用ctrl+k+第二次ctrl+d（效果不是很好，须先选中在使用）

第二种：下载Beautify插件，然后在命令面板中 选择beautify file(可以绑定快捷键 默认绑定快捷键shift+ctrl+F)

一键搭建各类语言的学习环境

Code runner插件。装完后可以

选择右上角，或者ctrl+alt+n可以运行输出。

选择文件的语言 先ctrl+k再按m(右下角默认纯文本的个选择语言)。

和chrome联动调试

下载Debugger for Chrome插件。

选择Debug的配置，有chrome: Attach和Launch两种模式。

  • 使用launch模式。需要修改的是等下调试所用的url，还有webRoot是等下所运行程序的文件夹，如${workspaceRoot}/src,设置断点，在弹出的页面上点击触发就会停在断点。

  • 使用attach模式，这个模式要设置chrome浏览器开发端口。

参考 https://www.cnblogs.com/pugang/p/5681325.html

自动检测代码规范和错误

安装 ESLint 并运行 npm install -g eslint

然后运行 eslint --init然后会有一些问题，回答完问题，会生成.eslintrc.js文件。（里面可以更改问题的答案）

右下角出现ESLint模式。
